# Posts Application

## Pre-requisites

-  Install Node.js version 20.17.0

## Getting started

-  Clone the repository

```
git clone <url>
```

-  Install dependencies

```
cd <project_name>
npm install
```

-  To start development, run:

```
npm run dev
```

-  To check eslint & typescript types tests, run:

```
npm run type-test
```
